

程序作用于IEEE33配电网中，通过接入风光新能源与储能元件，在考虑到新能源的不确定性与节点的需求响应的同时，实现了节点电压稳定和运行成本最小化的目标。

main.m为主程序，赋予子程序untitled2在不同场景下的风光数据。

untitled2.m为核心程序，基于主程序给予的风光数据，储能元件数据以及IEEE33的网络信息，以储能充放电功率与节点可转移负荷为主要变量。在满足潮流约束、储能约束、可转移负荷约束的条件下，实现了电压稳定；完成购电成本、损失成本、可转移负荷成本总和的最小化

datap.m 程序，基于拉丁超立方采样，实现在典型场景下的风光聚类
